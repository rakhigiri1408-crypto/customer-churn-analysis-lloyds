# customer-churn-analysis-lloyds
Customer churn prediction project using EDA and machine learning

This project focuses on predicting customer churn and identifying key factors influencing customer retention.

# Data Analysis
Performed exploratory data analysis (EDA)
Analysed customer demographics (age, gender, income)
Identified trends and patterns related to churn
# Data Preprocessing
Cleaned dataset
Encoded categorical variables
Standardised numerical features
# Model
Used Random Forest for churn prediction
Evaluated model using confusion matrix and performance metrics
# Key Insights
Age is the most important factor affecting churn
Income and marital status also influence customer behaviour
# Business Impact
Helps banks identify high-risk customers
Supports targeted retention strategies
